# Dvd Rental SQL project
## Overview

Used sql to perform analysis on a dvd rental database using a detailed and summary table. The goal was to filter for details on payments made only in cities in the US for the detailed table and the total revenue for each city.

SQL skills demonstated: table creation, joins, user-defined functions, triggers, and stored procedures

## Features
- Created a detailed table using multi layered joins, selecting only for cities in the US.
- Created a summary table that gives the total revenue from each city.
- Created trigger that updates summary table based on detailed table inserts.
- Wrote a stored proceture that turncates and refreshes the detailed and summary table.
